What is a set?

A set is a well defined collection of distinct objects, called elements or members of the set. A set can be represented in two forms. They are

  1. Roster/Tabular form
    2.
  2. Rule/Set builder form
    3.

What is Roaster and Rule method?

  • In Roster method, elements of the set are listed within the pair of brackets { } and are separated by commas.
    • Eg: R={2,4,6,8,10,12,14}

  • In Rule method, Set is represented by using statements depicting the relation among its elements.
    • Eg: S={x|x is an even number and x<15}

What is a venn diagram?

A Venn diagram is an illustration that uses circles to show the relationships among things or finite groups of things. Circles that overlap have a commonality while circles that do not overlap do not share those traits.
