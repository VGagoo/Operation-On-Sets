Difference of Sets

If there are two sets A and B, then the difference of two sets A and B is equal to the set which consists of elements present in A but not in B. It is represented by A-B.This operation can be represented as
A - B = {x : x ∈ A and x ∉ B}
We can also say, that the difference of set A and set B is equal to the intersection of set A with the complement of set B.

Eg: If set A = {1,2,3,4,5} and B = {4,5,6,7} Then, Difference of sets, A - B = {1,2,3}
